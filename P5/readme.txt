Se ha creado un objeto de la clase camara en la escena a la que se le aplica varios efectos.

Zoom--> para aplicar el zoom he optado por continuar con las teclas que hasta ahora estabamos usando
        Page_up para aumentar y Page_down para alejar.

Cambio entre perspectivas--> Se hace presionando la tecla 'O', con ella cambiaremos el valor booleano Ortho
                             pasando de perspectiva a ortogonal. 
                            Para apreciar bien el cambio se recomienda alejar la camara despues del presionado

PickColor--> Para apreciar este efecto basta con clickear con el boton izquierdo sobre cualquiera de las tres figuras 
             y el color pasa de azul a rojo y de rojo a azul si se vuelve a clickear.


Giro de la camara con el raton--> Basta con clickear con el boton derecho y dejarlo pulsado mientras giramos sobre la pantalla.
